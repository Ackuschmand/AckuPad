# AckuPad

Programmable Makropad based on a Seeed XIAO RP2040 with 6 switches, a toggle switch, an OLED display and a rotary encoder!

---

## Bill of Materials

### Electronics
- 1× Seeed XIAO RP2040  
- 7× MX-Style switches  
  (6× normal buttons, 1× mode button)
- 1× EC11 Rotary encoder  
- 1x 0.91 inch OLED displays  
- 1× PCB  

---

### Mechanical
- 1× 3D printed case (top)  
- 1× 3D printed case (bottom)
- 1× 3D printed case (middle)
- 1× Encoder knob
- 7× Blank DSA keycaps
- 4× M3x16mm screws
- 4× M3x5mx4mm heatset inserts
---

## Notes

- The mode button is a **normal mechanical keyboard switch**
- Buttons and encoder are **PC-controlled**
- Firmware does not contain fixed macros
- Case is fully 3D printed
- Custom PCB is used

---

## Files

- `RP2040 Code.ino` – firmware  
- `Test Script.ahk` – PC script  
- PCB and CAD files are in their respective folders
